This app is publicly available at https://jam373-cs3320-p5.southcentralus.cloudapp.azure.com/

Some notes on using the app at this URL:
We recommend using Firefox. This app has been tested and proven to work on Firefox.
Chrome also seems to work. However, we do NOT recommend using Safari - this app may not work in Safari.

When using this app at the URL above, please allow websocket connections several seconds to establish themselves.
After a page with websockets loads, websockets may disconnect and reconnect several times before establishing
a stable connection. This may take up to 10-15 seconds, so please be patient.
Once a stable connection is established, performance should be smooth.
